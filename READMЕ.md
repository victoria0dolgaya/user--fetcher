User Fetcher Application

A React-based application that fetches random user data from a public API and displays current weather conditions based on each user's location.
The project consists of two main routes:

Home - fetches users, displays cards, allows saving to localStorage and viewing weather.

Saved Users - displays locally saved users, allows deletion and exporting to Google Sheets.

Live Demo: https://user-fetcher-app-git-main-victoria0dolgayas-projects.vercel.app
